Set summary overlay — style mockups

Static HTML previews of four candidate layouts for the new "Set summary" overlay (replaces the scoreboard between sets during streaming). Open each file directly in a browser — they are self-contained, no JS or server required. Pick one (or ask for tweaks) before the runtime implementation moves forward.

card.html

Card minimalista

Header "Set 2 · 4:32", score grande centrado, chart pequeño debajo, dos columnas con stats clave.
hero.html

Hero score + chart de fondo

Score final 25–22 con tipografía protagonista, chart como marca de agua translúcida, footer con 4 stats.
timeline.html

Timeline storyboard

Chart de puntos a ancho completo como protagonista, pill superior con el score y banda inferior con eventos clave.
split.html

Split brand

Dos mitades verticales tintadas con los colores de cada equipo, logo + score + stat por lado, chart compacto en el centro.
All four mockups share the same demo data so the comparison stays apples-to-apples: Thunder Wolves 25 – Solar Hawks 22, Set 2, duration 4:32, longest streak 6 (TW from 12–12 to 18–12), timeouts 0 / 2, total points 47. Team colours are red #d4314c and gold #f0a020.
