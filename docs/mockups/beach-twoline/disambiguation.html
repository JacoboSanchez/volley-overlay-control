¿Qué marcador es de qué barra? — opciones

El caso difícil: modo simple sin icono, donde las dos barras quedan muy simétricas y el marcador alto (que abarca ambas) no deja claro a cuál pertenece. Local arriba (azul), visitante abajo (naranja). Compara las técnicas y dime cuál prefieres (o combina). El saque es ortogonal: aquí no se muestra para aislar la pista.
