Beach overlay — variante "dos líneas" (mockup)

De izquierda a derecha: marcador local (alto = las dos barras), las dos barras apiladas (arriba local, abajo visitante) y el marcador visitante (mismo alto). Cada equipo agrupa toda su info junto a su propio marcador, en orden de más próximo a más lejano del marcador: sets (números grandes) → icono → saque → nombre. Así el nombre queda en el borde interior y nada del rival aparece al otro lado de tu marcador. Tipografía y paleta heredadas de beach.css. Preview estático para validar la composición antes de implementarlo.

Notas de implementación (pendiente de tu OK): serían un template nuevo beach_twoline.html + CSS beach_twoline.css autodescubiertos por el motor de overlays (como el resto de estilos). Reutilizaría el mismo fitBeachNames() de app.js para ajustar el ancho de barra al nombre más largo (los dos lados simétricos), y los mismos resaltados por color de equipo. La altura de cada tarjeta de marcador se calcula como 2·barra + gap para cuadrar exactamente con las dos barras. Los sets/saque mostrados son opcionales: dime si los quieres dentro o los dejo fuera para una composición más limpia.
